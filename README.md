## Getting Started

### Installing
```bash
yarn install
```

### Run project
```bash
# For environment Dev
yarn start:dev
# Each environment Staging
yarn start:staging
# Each environment Prod
yarn start:prod
```

### Demo
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
